# Kamal's Server

This is the stuff I run on my personal server at home. It's a little Ubuntu box that never sleeps — hosts my cloud storage, a private search engine, some game servers, and all the automation glue that makes life easier.

This site is mostly for friends and family to see what I've been building. Technical documentation lives elsewhere.

## At a glance

**What I use the server for:**

- ☁️ **Cloud storage** — my own Nextcloud, no big tech involved
- 🔍 **Private search** — SearXNG, no tracking
- 📰 **RSS reader** — FreshRSS for following blogs and news
- 🎮 **Game servers** — retro arcade + multiplayer FPS
- 🤖 **AI agent** — Hermes Agent automates stuff for me
- 📸 **Photo gallery** — PiGallery2 for my photos
- 🔔 **Push notifications** — ntfy pings my phone
- 💾 **File sync** — Syncthing keeps devices in sync
- 🗂️ **File management** — web-based file browser
- 📋 **Dashboard** — Homepage showing everything at a glance
