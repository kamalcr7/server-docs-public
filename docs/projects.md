## BlockOps — Multiplayer FPS

A browser-based voxel-style multiplayer first-person shooter. Built with Node.js and Socket.IO for real-time networking. Friends can hop in from a browser — no install needed.

**URL:** `games.kamalrajnaidu.com/blockops/`  
**Tech:** Node.js, Socket.IO, Three.js

## Emulator Arcade — Retro Games

A retro game emulation arcade playable in any browser. Uses EmulatorJS with RetroArch compiled to WebAssembly — the emulation runs entirely on your device (no server-side rendering).

**URL:** `games.kamalrajnaidu.com/`  
**Tech:** Node.js, EmulatorJS, RetroArch WASM
